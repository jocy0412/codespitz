Iterable // Iterator 자바스크립트의 인터페이스는 특정 개체의 특정 키나 값을 정의한 것, Object의 형태를 인터페이스라고 함. 이터러블 인터페이스는 이터레이터라는 메소드를 갖고있다. 이터레이터는 키안에 함수가 들어가 있어야하고 함수는 이터레이터를 return 해야한다. 이터레이터 - next라는 함수를 호출하면 object를 return한다는 인터페이스를 정의함 next를 호출하면 value와 done 키가 있는 object를 return done - true /false (앞으로 계속 next를 호출 할 때 false) value - 이번에 받을 값이 있음, done이 true 이면 value는 undefined 제너레이터 - 유사 이터러블 - 제너레이터 자체는 이터러블 아니다. 이터러블은 이터레이터라는 함수를 호출해야 이터레이터 함수를 줌 - 제너레이터는 그냥 호출하면 이터레이터가 튀어나옴 suspend, resume (멈춤과 진행)
